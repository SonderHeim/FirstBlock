1. JVM - виртуальная машины, в задачу которой входит исполнение байткода, который формируется javac в JDK. Входит в состав JDK и JRE.
JRE - окружение для запуска Java программ. В состав входит JVM и библиотеки, необходимые для запуска байткода.
JDK - окружение для разработки приложений, содержит стандартные библиотеки, компилятор javac, JRE и т.д.
Разработчику нужен JDK, пользователю JRE.
2. Байт-код - это код Java скомпилированный javac в JDK. Байт-код исполняется виртуальной машиной JVM.
Из-за того, что пользователю на его компьютере необходимо иметь только виртуальную машину, которая самостоятельно посылает команды процессору,
можно воспроизвести единожды написанную программу на любой ОС под которую можно установить JRE.
